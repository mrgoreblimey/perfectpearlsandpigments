05 / Notes

What's Different in Elementor

Things the React design does that Elementor handles via different mechanisms.

Sticky header

Use the header Container's Motion Effects → Sticky → Top and set Effects Offset to 0. The kit's header already has a CSS class hook (ppp-sticky-header) if you want to add a background-change-on-scroll effect.

Card hover lift

The custom CSS class ppp-card ships globally — adds translateY(-5px) + soft shadow on hover. Apply it to any Container.

Cart drawer

Elementor Pro's Off-Canvas widget triggered from the header cart icon → embed the WooCommerce Cart widget inside it. Or use a plugin like CartFlows/Cart Lift for the slide-in.

Hero entrance animation

Each child widget supports Advanced → Motion Effects → Entrance Animation. Set the eyebrow/title/text/buttons to Fade in Up with staggered delays (0.1s / 0.25s / 0.4s / 0.55s) to recreate the cascade.
